*******************************************************
*  Description of the program
*******************************************************

This program will display the system information of the
machine running the program, including Systemname,
Nodename, Release, Version, Machine, LastBoot,
Uptime, UserModeTime, SystemModeTime, and Memory usage.

*******************************************************
*  Source files
*******************************************************

Name:  lab1.cpp
   Main program. It contains 3 functions. One to read in the line and parse the required data
   out of it, another to do the same thing but with a stopping point labeled, and the main
   function which is broken into part a and b. Part a prints out the system information, and
   part b pulls in data from the proc directory and parses it, calculates the output times/values,
   and prints them out as well.
